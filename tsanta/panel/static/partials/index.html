Добро пожаловать на EduJudge!

EduJudge - это платформа с разнообразными задачами на программирование. Её особенность в том, что помимо автоматических тестов, программы проверяет преподаватель.

Как работать на платформе

  1. На странице списка задач представлен список решенных и нерешенных задач. Выбирайте любую и в бой.
  2. Решать задачи можно на языке Python. В качестве средства для ввода исходных данных используется стандартный ввод. Рассмотрим работу на примере решения элементарной задачи сложения двух чисел
a, b = map(int, input().strip().split(' '))
print(a + b)

Данная программа:

  • считывает одну строку с помощью функции input
  • убирает лишние символы с обоих концов строки с помощь strip
  • создает список из двух элементов по разделителю пробел: split(' ')
  • приводит строки к типу int с помощью функции map
  • выводит результат в стандартный вывод с помощью print
